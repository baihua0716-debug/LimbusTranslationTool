# 零协汉化修正工具

这是一个本地运行的小工具，用来检索、修改并保留《边狱巴士公司》零协汉化 JSON 的玩家自定义修正。

## 启动

```powershell
.\start.ps1
```

或直接运行：

```powershell
python .\app.py
```

默认会打开一个桌面应用窗口。点击“选择文件夹”选择游戏汉化目录。可以选 `Lang`，也可以直接选 `Lang\LLC_zh-CN`。如果文件夹选择窗口没有弹出来，也可以手动输入路径后点击“绑定”。

## 能做什么

- 扫描所有 JSON 文本字段，并按剧情、技能、被动、E.G.O、语音、异想体、敌人、关键词、物品、界面等分类。
- 对人格技能、被动、E.G.O 技能、语音等内容尽量按 12 位罪人细分。
- 搜索译文、文件名、字段名、条目 ID，并在右侧直接修改。
- 支持批量替换某个名词翻译，可先预览影响的文件、字段和示例。
- 支持撤销最近一次写入操作；撤销前同样会备份当前 JSON。
- 每次保存前自动备份原 JSON。
- 玩家修改会记录在 `user_data\overrides.json` 和 `user_data\history.jsonl`。
- 零协更新覆盖文件后，重新绑定目录并点击“重放历史修改”即可把玩家修正写回。
- “备份当前目录”会把当前绑定目录下的 JSON 打成 zip。

## 数据位置

- 历史和修正：`user_data`
- 单文件写回备份：`user_data\backups`
- 完整目录备份：`user_data\backups\full_*.zip`

工具不会把历史文件写进游戏目录；只有点击“保存修改”或“重放历史修改”时，才会改动绑定的汉化 JSON。

## 打包成玩家版 EXE

在开发机运行：

```powershell
.\build_windows.ps1
```

构建完成后，把 `release\LimbusTranslationTool.zip` 发给玩家。玩家解压后双击 EXE 即可使用，不需要安装 Python，也不需要手动打开浏览器。
