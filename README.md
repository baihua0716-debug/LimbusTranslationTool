# 零协汉化修正工具

一个本地运行的 Windows 工具，用来检索、修改并保留《边狱巴士公司》零协汉化 JSON 的玩家自定义修正。

## 下载

在这里下载最新版：

[LimbusTranslationTool.zip](https://github.com/baihua0716-debug/LimbusTranslationTool/releases/latest/download/LimbusTranslationTool.zip)

解压后双击 `LimbusTranslationTool.exe` 即可使用。

## 使用方法

1. 打开工具后，点击“选择文件夹”。
2. 选择游戏里的汉化目录，可以选 `Lang`，也可以直接选 `Lang\LLC_zh-CN`。
3. 搜索要改的译文。
4. 在右侧编辑后点击“保存修改”。

工具会在写入前自动备份原 JSON。

## 主要功能

- 扫描所有 JSON 文本字段，并按剧情、技能、被动、E.G.O、语音、异想体、敌人、关键词、物品、界面等分类。
- 对人格技能、被动、E.G.O 技能、语音等内容尽量按 12 位罪人细分。
- 搜索译文、文件名、字段名、条目 ID，并直接修改。
- 批量替换某个名词翻译，可先预览影响的文件、字段和示例。
- 撤销最近一次写入操作，包括单条保存、批量替换、重放历史。
- 零协汉化覆盖更新后，可以“重放历史修改”把玩家修正写回新版 JSON。
- 可备份当前绑定目录下的 JSON。

## 批量替换

左侧“批量替换”里填写“原词”和“新译名”，先点“预览”，确认影响范围后再点“执行”。

勾选“使用当前筛选范围”时，只会替换当前分类、罪人、文件筛选、字段筛选范围内的命中项；取消勾选则会在所有 JSON 文本里替换。

## 撤销

点“撤销最近修改”可以回退最近一次写入操作。

如果某个字段在那之后又被改过，工具会把它当作冲突跳过，避免覆盖新的内容。

## 数据保存在哪里

工具会在 EXE 同目录生成 `user_data` 文件夹：

- `overrides.json`：你的修正记录
- `history.jsonl`：操作历史
- `backups`：写入前自动备份

想迁移到另一台电脑时，把 EXE 和 `user_data` 一起复制过去。

## 说明

这个工具会直接修改你绑定的零协汉化 JSON 文件。  
写入前会自动备份，但仍建议先关闭游戏再操作。
