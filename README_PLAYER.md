# 零协汉化修正工具 使用说明

## 最简单用法

1. 解压压缩包。
2. 双击 `LimbusTranslationTool.exe`。
3. 工具窗口打开后，点“选择文件夹”。
4. 选择游戏里的汉化目录，可以选 `Lang`，也可以选 `Lang\LLC_zh-CN`。
5. 搜索要改的译文，在右侧修改后点“保存修改”。

## 批量替换名词

左侧“批量替换”里填写“原词”和“新译名”，先点“预览”，确认影响范围后再点“执行”。

勾选“使用当前筛选范围”时，只会替换当前分类、罪人、文件筛选、字段筛选范围内的命中项；取消勾选则会在所有 JSON 文本里替换。

## 撤销改错

点“撤销最近修改”可以回退最近一次写入操作，包括单条保存、批量替换、重放历史。

如果某个字段在那之后又被改过，工具会把它当作冲突跳过，避免覆盖新的内容。

## 恢复零协原文

点“恢复零协原文”可以把本工具修改过的文本一次恢复为修改前的零协译文。恢复前会自动备份；已经被新版汉化或其他程序再次修改的文本会作为冲突跳过。

整次恢复可以通过“撤销最近修改”撤销。

## 游戏更新后

零协汉化更新会覆盖 JSON。更新后重新打开工具，绑定新的汉化目录，然后点“重放历史修改”。

工具会把你以前保存过的修正重新写回新文件；找不到对应位置的内容会跳过，不会乱改。

## 数据保存在哪里

工具会在 EXE 同目录生成 `user_data` 文件夹：

- `overrides.json`：你的修正记录
- `history.jsonl`：操作历史
- `backups`：写入前自动备份

想迁移到另一台电脑时，把 EXE 和 `user_data` 一起复制过去。

## 退出

点右上角“退出”，或直接关闭工具窗口。
